## Getting Started
### Setup Local Environment

1.  Install Git

**macOS/Linux**

In a terminal: 

```
user@computer:~$ sudo apt install git
[sudo] password for user:
``` 
Enter your password, note that characters may not be shown when typing. 

**Windows**

Install [Git for Windows](https://gitforwindows.org/)
